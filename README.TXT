# Smart Invoice Bot Pro — Client Instructions

Welcome! This tool allows you to generate and email invoices easily. 

## Requirements
- Windows PC
- Python 3.9+ installed (https://www.python.org/downloads/)
- Verified SendGrid account

## Setup & Usage
1. Extract the `smart_invoice_bot.zip` folder to your PC.
2. Open the folder and double-click `run_app.bat`.
3. The bot will open in your browser.
4. In the sidebar:
   - Enter your **SendGrid API Key**.
   - Enter your **Sender Email** (must be verified in SendGrid).
   - Enter your **Business Name** and **VAT %** if applicable.
5. Upload your CSV or Excel file with the following columns:
   - `ClientName`, `Email`, `Amount`, `ReferenceNo`
6. Check “Email each invoice automatically” if you want to send invoices directly.
7. Click “Generate All Invoices”.
8. Download all invoices as a ZIP file.

## Notes
- The bot does NOT include any personal API keys — use your own.
- Make sure your emails are verified in SendGrid to avoid errors.
